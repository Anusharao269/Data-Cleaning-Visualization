# Data Cleaning & Visualization Project

## Objective

Work with a raw dataset to clean, process, and visualize insights using
Python.

## Features

-   Handle missing values
-   Remove duplicates
-   Detect and remove outliers
-   Visualize insights using Matplotlib and Seaborn

## Files

-   raw_employee_data.csv → Raw dataset
-   data_cleaning_visualization.py → Python script for cleaning and
    visualization
-   project_notebook.ipynb → Notebook version of the project
-   requirements.txt → Required libraries

## How to Run

1.  Install requirements: pip install -r requirements.txt

2.  Run the script: python data_cleaning_visualization.py

## Output

-   cleaned_employee_data.csv
-   salary_distribution.png
-   department_count.png
-   age_salary_scatter.png
